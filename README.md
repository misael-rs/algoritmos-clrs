# algoritmos-clrs
Repositório onde irei colocar tentativas pessoais de implementação de algoritmos do livro de Algoritmos (CLRS)
O objetivo deste projeto é praticar a linguagem C por meio da implementação de algoritmos clássicos do livro.

# Tecnologias e Ferramentas
Linguagem: C
Compilador: GCC
Ambiente: Zorin OS (Linux)

# Como compilar e executar
Para rodar qualquer algoritmo localmente, abra o terminal e, na pasta do arquivo, digite:
gcc insertion-sort.c -o insertion-sort
./insertion-sort

O comando acima foi para o exemplo do algoritmo Insertion Sort, mas, para outros arquivos, basta substituir o nome pelo arquivo alvo.
